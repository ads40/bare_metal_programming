Xv6 is "a simple Unix-like teaching operating system".

Xv6 is available from the creators originally for x86 but more recently for
RISC-V:

- [https://pdos.csail.mit.edu/6.828/2023/xv6.html](https://pdos.csail.mit.edu/6.828/2023/xv6.html)

```sh
git clone https://github.com/mit-pdos/xv6-riscv.git
git clone https://github.com/mit-pdos/xv6-riscv-book.git
```

For older versions of the Raspberry Pi, there was a port of xv6 to `Armv7-A`
created at the University of Otago:

- [https://github.com/patha454/xv6_pi_mp](https://github.com/patha454/xv6_pi_mp)
